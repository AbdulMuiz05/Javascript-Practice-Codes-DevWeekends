# Javascript-Practice-Codes-DevWeekends
JavaScript practice programs covering fundamental concepts, DOM manipulation, functions, arrays, objects, events, and basic problem-solving exercises developed during the Dev Weekends learning track.

Practice Codes

1. Temperature Converter
Converts temperatures between Celsius and Fahrenheit using JavaScript functions and arithmetic operations.

2. Number Guessing Game
Implements a simple number guessing game using random number generation and conditional statements.

3. Simple Calculator
Performs basic arithmetic operations (addition, subtraction, multiplication, and division) using functions and switch statements.

4. Student Grade Calculator
Calculates and displays a student's grade based on marks using conditional logic.

5. To-Do List
Demonstrates basic task management by adding and removing tasks using JavaScript arrays and functions.

6. Background Color Changer
Changes the webpage background color dynamically using DOM manipulation and event handling.

7. BMI Calculator
Calculates Body Mass Index (BMI) and classifies the result into different health categories.

8. Digital Clock
Displays the current time in real time using the Date object and the setInterval() function.

9. Character Counter
Counts the number of characters and words entered by the user using string methods.

10. Random Quote Generator
Displays a randomly selected motivational quote from a predefined collection using arrays and randomization.

11. Palindrome Checker
Checks whether a given string reads the same forward and backward using string manipulation techniques.

12. Reverse a String
Reverses a given string using JavaScript string and array methods.

13. Count Vowels
Counts the total number of vowels in a given string using loops and conditional checks.
